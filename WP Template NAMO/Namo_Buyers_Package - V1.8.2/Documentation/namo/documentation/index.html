Thank you for purchasing my theme. This document has all the details to help you get started with NAMO and setup your website using our theme.

Getting Started

i. Upload NAMO

If you have not set up your WordPress installation yet, you can grab a copy of latest WordPress version here.

ii. Install and Activate The Theme

You can install and activate the theme in more than one ways as give below -

Go to Appearances > Themes > Add New . Clicking on Add New to add the theme will Install and Activate your theme OR You can Install the theme using any FTP program, by uploading the entire ‘NAMO’ folder (find it in your Downloaded File > wp-content > themes ) to the corresponding folder on your server ( http://yourdomain.com/wp-content/themes/ )To Activate the theme, login to your blog, and goto

Menu > Appearance > Themes.

Available themes will show ‘NAMO’. Click on ‘Activate’ button under NAMO.

iii. Recommended WordPress Settings

Comments [optional] – Goto Menu > Settings > Discussion and under ‘Other comment settings’ check the following two options:

Enable threaded (nested) comments
Break comments into pages

Importing Demo Content

Navigate to
Menu > Tools > Import

and click on ‘WordPress’

Updating the theme

1. Download the latest version of the theme by logging on to your Themeforest account  and then navigating to the "Downloads" section. 

2. Log on to your wordpress admin panel, and navigate to Appearances -> Namo Options -> Import / Export  and then download a copy of the options panel. This backup is to ensure that you can restore your options & settings incase something goes wrong during the theme update process. We strongly recommend doing this, though it is not mandatory

3. Then Navigate to Appearances -> Themes , delete the current version of the Namo and then install the latest theme zip you downloaded from themeforest in the first step. 

4. Alternatively you can extract the theme zip you downloaded in the first step, connect to your server via FTP and navigate to /wp-content/themes folder of your wordpress installation, replace the existing folder named "namo" with the extracted folder. 

Updating Revolution Slider Plugin

1. Please check the Namo changelog to see if there is an updated version of Slider Revolution plugin has been included. 

2. If so, download the complete theme package by logging on to your Themeforest account  and then navigating to the "Downloads" section and extract the zip file. 

 

 

3. In the package folder you will find the Documentation, Sample Content, Theme Zip, Child Theme Zip and revslider.zip . 

 

 

4. Login to your wordpress admin area, Navigate to the Revolution Slider Panel from the links in the left sidebar and find the "Manual Plugin Update" button in the right bott om corner of the screen. Click the button and upload & install the revslider.zip file from step-3 to update the plugin.

Page Builder Part 1 - Sections

The theme uses our custom Page Builder and is highly powerful.

Before you start using the Page Builder in your pages, please ensure that you have checked the 'Enable Page Builder' in the options panel in the location below - 

Options > Appearance > NAMO Options > General

This is very vital and your pagebuilder will not work if this is not checked. 

Sections, Rows and Columns are the basic elements that are used to construct a page using the Page Builder. A Page will be broken into one or more Sections. One Section can have one or more Rows and in turn, one Row could be formed with one or moreCoulmns. The module/actual content of a page, is written inside the respective Columns.

Before you get to construct your page with the Page Builder, please note that you have to Name the page and either save it (in drafts) or publish it. Without naming and saving/publishing the the page, the configuration done using the Page builder will not take effect on the Page.

As and when you make changes to the page using the Page Builder, the changes will be Auto Saved. However deletion and re-positioning of the modules will not be Auto-Saved. Please make sure you click the Page Builder 'Save' button in cases when you do not see the Changes reflecting in the Web Page.

Building a Single / One Page Website

Namo allows you to setup your site as a Single Page / One Page website. To do this,

1. Create a page in the backend and build it using the page builder.

2. Assign a unique ID to each section in its Settings. 

 

3. And then when you create the menu you need to assign the links in the format http://www.yourwebsite.com/#SECTION_ID . These links will then smooth scroll to its respective section. 

 

Blog Part 1 - Assign a Blog Page

The first step to set up a blog page, is to create an empty Page anew. Then, navigate to 

Options -> Settings -> Readings

In the dorp down called Posts Page, choose the Page that you created and want to be the Blog Page. 

Blog Part 2 - Main Posts Page Setting

There are a couple of options that you should be setting up from the Options Panel to customize the way your Main Posts page will look. 

Navigate to 

Options -> Appearances -> NAMO Options -> Blog Settings
  • Blog Style - We have offered three types of Blog Styles. Each of this Style is avialble in the Demo Site. Choose any one of the Style that you want. 
  • Blog Sidebar Position - You can choose to have a Right Sidebar or left Sidebar for your blog.
  • Blog Column - Number of columns in the Main Posts page. This is applicable only if you have chosen 'Style - 3 Masonry' for your blog. 

 

Blog Part 3 - Single Posts Page Setting

There are some settings provided in the Options Panel for Blog that will help you format the Header and Hero sections of your Sinlge Blog Post pages. Having said that, the Header and Hero Section for the Main Blog page should be set up in the respective page only.

Please note that this setting is not mandatory. 

Navigate to - 

Options -> Appearances -> NAMO Options -> Single Blog Hero section.

The options given here are for the Hero section of all the Single Blog Post pages. 

  1. Header Hero Section from - This is the most Important criteria. If you choose 'Options Panel - Here' option, the Hero section option that you set up in this page will appear for all the Single Blog post pages. On the otherhand, if you choose 'Single Page', the Hero section setup in the individual Blog post page will be shown. The rest of the Hero section configuration is similar to what is given under the title Pages Part 2 - Header and Hero Section

Shop Part 1 - Assign a Shop page

We use WooCommerce for shop in our theme. 

Firstly, creat a Page anew that you want to assign as the Shop page. this can just be an empty Page. Then, follow the steps below to install Woo commerce plug in and set up the Shop in NAMO

  1. The first thing you should do is download the Woo-Commerce plugin
  2. You will be able to see a 'Pink Color' notification that reads Install Woo Commerce Pages once the plugin is in place. Click on this notification and install. Clicking the Install Pages option, will create the Shop, Cart, My Account, Check Out and other Shop related pages
  3. The next step, would be to Select the Shop Page. In the options panel, Go to 
Woo Commerce > Setting > Products Tab > Product Archive / Shop Page .

Pick the Shop page from this drop down, as shown in the image below - 

Shop Part 2 - WooCommerce Settings

Once you have set up the Shop Page, you will have to make a couple of settings in the Options Panel to govern the way your Shop page will look. 

Navigate to - 

Options > Appearance > NAMO Options > Shop
  • Shop With Sidebar? - Check this option if you want a Side bar in your shop page. By default, if enabled this will bring the Shop Sidebar. 
  • Number of products per page - Give the number of Products you want to show in the Main Shop Page.  

Shop Part 3 - Single Shop/Product Page Setting

There are some settings provided in the Options Panel for Shop that will help you format the Header and Hero sections of your Sinlge Shop/Product pages. Having said that, the Header and Hero Section for the Main Shop page should be set up in the respective page only.

Please note that this setting is not mandatory. 

Navigate to - 

Options -> Appearances -> NAMO Options -> Single Shop Hero section.

The options given here are for the Hero section of all the Single Blog Post pages. 

  1. Header Hero Section from - This is the most Important criteria. If you choose 'Options Panel - Here' option, the Hero section option that you set up in this page will appear for all the Single Shop/Product pages. On the otherhand, if you choose 'Single Page', the Hero section setup in the individual Product page will be shown. The rest of the Hero section configuration is similar to what is given under the title 'Pages Part 2 - Header and Hero Section'

Contact Page

This section explains how to set up your contact page. 

  • Navigate to 
Options > Appearance > NAMO Options > Contact Settings

In this page provide the email address. This is essential for your Contact Page.

  • Navigate to 
Options > Pages > Add New

Name the page and choose the 'Contact' page template for this page. By assinging the Contact page template, the theme populate this page with a contact form, and the email will be sent to the email id that was provided in the Contact Settings in Options Panel (above). All other details that you need on the contact page, can be  included using the Page BUilder.
